# Design Tokens Report

Derived from textual requirements.

## Colors
| Token | Value | Usage |
|---|---|---|
| Primary | `#C23B4A` | Main CTAs, Branding, Active States |
| Secondary | `#6B8E23` | Accents, Secondary Actions, Success Indicators |
| Accent | `#FFD700` | Highlights, Badges |
| Neutral-50 | `#F8F8F8` | Backgrounds |
| Neutral-100 | `#EEEEEE` | Borders, Dividers |
| Neutral-200 | `#DDDDDD` | Disabled States |
| Neutral-800 | `#333333` | Primary Text |
| Neutral-500 | `#888888` | Secondary Text |
| Success | `#4CAF50` | Operational Success |
| Warning | `#FFC107` | Cutoff Warnings |
| Error | `#F44336` | Validation Errors, Zone Failure |

## Typography
*Assumed Font Family:* `Montserrat` (Headings), `Open Sans` (Body).

| Token | Size / Height | Weight | Usage |
|---|---|---|---|
| H1 | 48px / 56px | Bold | Page Titles, Hero Text |
| H2 | 36px / 44px | SemiBold | Section Headers |
| H3 | 28px / 36px | Medium | Card Titles, Subsections |
| Body | 16px / 24px | Regular | Main Content |
| Caption | 14px / 20px | Regular | Hints, Metadata |

## Spacing & Shape
- **Base Unit:** 8px
- **Border Radius:** 8px (Small), 12px (Medium), 16px (Cards/Large)
- **Shadows:** `0px 4px 6px rgba(0, 0, 0, 0.1)` (Subtle depth)

## Grid System
- **Desktop:** 12 cols, 24px gutter, Max width 1200-1320px
- **Tablet:** 8 cols
- **Mobile:** 4 cols
